\begin{problem}

How many simple graphs are there with vertex set $\set{1,2,\dots,n}$?\hfill \examrule[0.7in]

\begin{solution}
There are $\binom{n}{2}$ potential edges, each of which may or
may not appear in a given graph.  Therefore, the number of graphs is:
\[
2^{\binom{n}{2}}
\]
\end{solution}

\end{problem}
