\begin{problem}
Truth values for propositional variables $P,Q,R$ are chosen
independently, with $\pr{P=\true} = 1/2, \pr{Q=\true} = 1/3,
\pr{R=\true} = 1/5$.  What is
\[
\pr{P \QIMPLIES (Q \QIMPLIES R) = \true}\ ?
\]
